# MySQL Setup Instructions for FitSphere Project

## Option 1: Install XAMPP (Recommended for Windows)

1. Download XAMPP from: https://www.apachefriends.org/download.html
2. Run the installer and select components: Apache, MySQL, PHP (minimum required)
3. Complete the installation
4. Launch XAMPP Control Panel
5. Start Apache and MySQL services
6. Update your .env file with:
   ```
   PORT=5000
   JWT_SECRET=some_random_secret
   MYSQL_HOST=localhost
   MYSQL_PORT=3306
   MYSQL_USER=root
   MYSQL_PASSWORD=  (leave empty if no password is set for root)
   MYSQL_DATABASE=fitsphere
   ```

## Option 2: Install MySQL Community Server directly

1. Visit: https://dev.mysql.com/downloads/mysql/
2. Download MySQL Community Server
3. Run the installer and follow the setup wizard
4. During setup, note the root password you create
5. Update your .env file with the correct root password

## After MySQL Installation:

1. Open Command Prompt as Administrator
2. Navigate to the project directory:
   ```
   cd "C:\Users\SUMIT\OneDrive\Desktop\FitSphere\project"
   ```
3. Create the database using one of these methods:

   Method A: Using MySQL command line
   ```
   mysql -u root -p
   # Enter your password when prompted
   # Then run:
   CREATE DATABASE fitsphere;
   USE fitsphere;
   # Paste the contents of database.sql file
   ```

   Method B: If you can connect to MySQL from the project directory:
   ```
   mysql -u root -p -e "CREATE DATABASE IF NOT EXISTS fitsphere;"
   mysql -u root -p fitsphere < database.sql
   ```

4. Start the application:
   ```
   npm start
   ```

## Troubleshooting Common Issues:

1. If you get "access denied" errors, verify your MySQL root password
2. If you get "connection refused" errors, ensure MySQL service is running
3. If you get port conflicts, try changing the port in .env file
